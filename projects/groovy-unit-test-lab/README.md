# Groovy Unit Test Lab

A tiny Groovy hands-on for script-style unit testing with plain `assert`.

## Run With Groovy

```bash
groovy projects/groovy-unit-test-lab/CalculatorTest.groovy
```

## Environment Check

This command is safe to run even when Groovy is not installed:

```bash
python3 projects/groovy-unit-test-lab/check_groovy.py
```

## Exercise

1. Add a failing assertion.
2. Add a new method to `Calculator`.
3. Port this tiny test to JUnit, Spock, Gradle, or Bazel.
