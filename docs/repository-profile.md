# Repository Profile: learning-build-systems

Last verified: 2026-06-20

## GitHub Description

Build systems learning repo for Bazel, Nix, dev shells, and local CI parity.

## Topics

- `learning`
- `bazel`
- `nix`
- `build-systems`
- `developer-experience`

## Audience

Engineers who want to understand when Bazel, Nix, language-native tools, and command runners help or hurt.

## First Milestones

1. Add a small Bazel Java build and test.
2. Add a small Bazel Go build and test.
3. Add a TypeScript Bazel example and compare it with pnpm.
4. Add Nix dev shells for Node, Python, and Java.
5. Add GitHub Actions templates for Bazel and Nix.

## Public Safety Checklist

- No machine-specific absolute paths.
- Nix examples pin inputs or document why they are not pinned.
- Bazel examples document build and test commands.
- CI examples avoid secrets unless they use clear placeholders.
